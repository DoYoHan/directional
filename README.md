## 프로젝트 실행 방법 (Getting Started)

이 프로젝트는 Next.js + React + TypeScript 기반으로 구성되어 있으며, 다음 단계에 따라 로컬 환경에서 실행할 수 있습니다.

1. 패키지 설치

프로젝트에 필요한 Node 패키지를 설치합니다.
- npm install

2. 실행

아래 명령어로 Next.js 서버를 실행합니다.
- npm run dev

3. 접속

실행 후 브라우저에서 아래 주소로 접속합니다.
- http://localhost:3000


## 주요 구현 기능

1. 게시판 기능
- 게시글 작성 / 조회 / 수정 / 삭제 (CRUD)
- 게시글 목록 커스텀

2. 데이터 시각화 기능
- Mock API를 활용한 데이터를 시각화 (바, 도넛, 스택 등등)
- 차트 색상 커스텀
